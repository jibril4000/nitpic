Two minutes to your first nitpic

nitpic sends your clicks and comments straight into a running Claude Code session. The extension is installed — now connect the Claude Code side.

✓

Chrome extension installed

That's this. Nothing more to do here.

2

Make sure you have one of the following coding tools

Claude Code Cursor Codex Gemini CLI
3

Install the nitpic plugin

Paste these into any Claude Code session:

/plugin marketplace add jibril4000/nitpic
/plugin install nitpic@nitpic
4

Type /nitpic in your session

That session becomes the one your feedback lands in. Run it in a different session anytime to move the connection.

5

Click the nitpic icon on any page

The floating panel opens and pairs itself automatically — nothing to copy. Then hit Start feedback and click anything.
